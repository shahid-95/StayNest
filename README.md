# StayNest – Hotel Booking Website

## 📌 Overview

**StayNest** is a full-featured hotel booking web application inspired by Airbnb. Users can explore hotels, view detailed property information, read and post reviews, and securely authenticate to manage their bookings.

## 🚀 Features

* 🔍 Search hotels by location
* 👤 Secure user authentication (Login, Signup, Logout)
* 📝 Add reviews and ratings
* 📸 Image gallery for each hotel
* 🗄️ Data stored in MongoDB Atlas
* 🌐 Deployed on Render

## 🛠 Tech Stack

**Frontend:** HTML, CSS, JavaScript
**Backend:** Node.js, Express.js
**Database:** MongoDB Atlas
**Deployment:** Render

## 📂 Project Setup

```bash
# Clone the repository
git clone https://github.com/Mohd-Shahid-95/StayNest-hotel.git

# Navigate to project folder
cd StayNest-hotel

# Install dependencies
npm install

# Start the server
npm start
```
